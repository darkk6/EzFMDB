## 放置 FileMaker for PHP Library

你可以在底下位置找到 Library 檔案：
```
{FileMaker Server安裝路徑}/Web Publishing/FM_API_for_PHP_Standalone.zip
```
直接將其內容解壓縮到此目錄中即可，完成後檔案結構應該會是：

	fm_api/
	|---README.md (這個檔案)
	|---FileMaker.php
	|---FileMaker/
		|---Command.php
		|---Error.php
		|---(其他資料夾與檔案...)
---
